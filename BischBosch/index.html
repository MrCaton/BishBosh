Bish-Bosh 2.0

Original game

Bish-Bosh lists all numbers between 1 and 100, but some exceptions:

  • If the number is evenly divisible with 3 listed ‘Bish’ instead of the number.
  • If the number is evenly divisible by 4, ‘Bosh’ is listed instead the number.
  • If the number is evenly divisible by both 3 and 4, write "Bish-Bosh" instead of the number.

The game

The new game is similar to the origininal but with some variation. First of all you can choose how many numbers you want to show (default 100). Now you can choose what numbers you want to display Bish and to display Bosh.
